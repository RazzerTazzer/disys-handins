Start server with command: go run server/server.go

to join a client write the command: go run client/client.go -name SomeName -port SomeInt
example:  go run client/client.go -name Allan -port 4400

to send message just write in the commandline and hit enter. To leave the chat, just write 'leave'

have as many clients join as you want, just make sure they have a unique port
